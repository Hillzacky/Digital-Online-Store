Axies – you can create your own download script where users can download PSD’s, Zip files, and any other kind of graphic file from your site. Pixel was built on top of Laravel. Checkout the demo to learn more.

Axies is unique and modern Digital Downloads script Template made for Online Digital Shops & Creative people. It’s fully responsive and easy to customize. It has unique design which keeps up with web-desing trends. Dizital script can also be used photographers and Elements are easy to change according to requirements.

It’s also 100% responsive that’s why it will work nicely on all smart devices(smart phones, tablet, PCs and desktops). It’s also well documented and clean coded that’s why anyone can change it easily.

Installation

Important Programs

Important programs must be installed for the script to work with you

  • Localhost Server
  • Program for extract Axies.zip Files
  • Step 1: Run Your local server Apache and MySQL
  • Step 2: open link Phpmyadmin to step database Phpmyadmin
  • Step 3: Create database name Axies.
  • Step 4: Extract Axies.zip File to Desktop.
  • Step 5: import database From Files Database. (Axies.sql).
  • Step 6: run your Project: php artisan serve
  • Enjoy your project
  • See This video To learn Installation App Installation Video

Database

Note : if you want to change database name go to .env file and change name:

APP_DEBUG=true
APP_URL=http://localhost:8000

LOG_CHANNEL=stack

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=Axies
DB_USERNAME=root
DB_PASSWORD=

BROADCAST_DRIVER=log
CACHE_DRIVER=file
QUEUE_CONNECTION=sync
SESSION_DRIVER=file
SESSION_LIFETIME=120
